# Tax Clarity – Tax & Accounting Services Website

## Overview

Tax Clarity is a professional, responsive business website developed to provide comprehensive tax, accounting, auditing, GST, and business consulting services. The website is designed with a modern user interface and user-friendly navigation to help clients easily explore services, submit inquiries, and connect with tax professionals.

This project demonstrates front-end web development skills using HTML5, CSS3, Flexbox, Grid Layout, and Responsive Design principles.

## Key Features

- Professional and modern business website design
- Fully responsive layout for Desktop, Tablet, and Mobile devices
- Interactive service showcase section
- Client registration and inquiry forms
- About Us section highlighting company expertise
- Customer testimonials section
- Integrated Google Maps location
- Font Awesome icons for enhanced user experience
- Clean and structured user interface
